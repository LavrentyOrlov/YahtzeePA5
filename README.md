# Project Name: Yahtzee

## Project Info
- Date: October 2025
- Created by: Lavrenty "Larry" Orlov

## Overview
This is a text-based version of a dice game called Yahtzee involving features like handling user input in Command Prompt,
logical decision-making, checking for valid user input, numerical combination calculations, and random number generation.

## Features
- Controls the complete gameplay of Yahtzee combined with a main menu, displaying rules, and giving the player the option to leave the game.
- Modular design with separate '.c' and '.h' files
- Easy to run in Visual Studio

## Tech Stack
- Language: C
- IDE: Visual Studio Community

## How to Run
1. Clone the repository on your device's command-line interface:
- ```git clone https://github.com/LavrentyOrlov/YahtzeePA5.git```
2. Open the project in Visual Studio.
3. Build the solution.
4. Run the program.
